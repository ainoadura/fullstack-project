# Fullstack Project - [Nombre pendiente]

Este es un proyecto profesional desarrollado con **React**, **TypeScript**, **Tailwind CSS** y una **API propia**. La aplicación permite catalogar y compartir libros, películas y series en un entorno privado entre amigos.

## 📁 Estructura de Documentación
En este repositorio se documenta todo el ciclo de vida del desarrollo:

*   [**Metodologías Ágiles**](./docs/agile.md): Investigación sobre Scrum, Kanban y la metodología elegida.
*   [**Idea del Proyecto**](./docs/idea.md): Definición del problema, usuario objetivo, funcionalidades MVP y mejoras futuras.

## 🚀 Tecnologías Principales
- **Frontend:** React + Vite + TypeScript.
- **Backend:** Node.js + Express (Arquitectura por capas).
- **Base de Datos:** PostgreSQL / MongoDB (vía Prisma/Mongoose).
- **Estilos:** Tailwind CSS.

## 🛠️ Arquitectura
El proyecto sigue una estructura de **API REST** con un cliente tipado en el frontend, asegurando la integridad de los datos desde la base de datos hasta la interfaz de usuario.
